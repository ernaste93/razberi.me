Прогресии

Аритметична · Геометрична · Безкрайна · Суми · ~18 мин
1. Какво е прогресия?

Прогресия е редица от числа, в която всеки следващ член се получава по определено правило. Матурата изисква да познаваш два вида: аритметична и геометрична.

Означения: членовете на прогресията се означават с a₁, a₂, a₃, ..., aₙ (за аритметична) или b₁, b₂, b₃, ..., bₙ (за геометрична).

Аритметична прогресия
Всеки член = предишен + d
Разлика d = const
Пример: 2, 5, 8, 11, ... (d = 3)
Геометрична прогресия
Всеки член = предишен × q
Знаменател q = const
Пример: 2, 6, 18, 54, ... (q = 3)
2. Аритметична прогресия

В аритметичната прогресия разликата между всеки два последователни члена е постоянна и се нарича разлика d.

Формулата за n-тия член и за сумата на първите n члена:

n-ти член на аритметична прогресия aₙ = a₁ + (n − 1) · d
Сума на първите n члена (две еквивалентни формули) Sₙ = n · (a₁ + aₙ) / 2
Sₙ = n · (2a₁ + (n − 1) · d) / 2
💡
Как да разпознаем АП: Изчисли разликата между последователни членове: a₂ − a₁, a₃ − a₂, ... Ако разликите са еднакви – това е аритметична прогресия.
📐
Средно аритметично свойство: В аритметична прогресия всеки вътрешен член е средното аритметично на съседните: aₙ = (aₙ₋₁ + aₙ₊₁) / 2.
📌
Разликата d може да е: положителна (нарастваща редица), отрицателна (намаляваща редица) или нула (константна редица).
Пример 1: Дадена е аритметична прогресия 3, 7, 11, 15, ... Намери a₁₀ и S₁₀.
1
a₁ = 3, d = 7 − 3 = 4
2
a₁₀ = 3 + (10 − 1) · 4 = 3 + 36 = 39
3
S₁₀ = 10 · (3 + 39) / 2 = 10 · 42 / 2 = 210
4
Отговор: a₁₀ = 39, S₁₀ = 210
Пример 2: Намери разликата d, ако a₁ = 5 и a₇ = 29.
1
aₙ = a₁ + (n − 1) · d → a₇ = 5 + 6d = 29
2
6d = 24 → d = 4
3
Отговор: d = 4
3. Геометрична прогресия

В геометричната прогресия отношението между всеки два последователни члена е постоянно и се нарича знаменател q.

n-ти член на геометрична прогресия bₙ = b₁ · qⁿ⁻¹
Сума на първите n члена (при q ≠ 1) Sₙ = b₁ · (qⁿ − 1) / (q − 1)
💡
Как да разпознаем ГП: Изчисли отношението между последователни членове: b₂/b₁, b₃/b₂, ... Ако отношенията са еднакви – това е геометрична прогресия.
📐
Средно геометрично свойство: В геометрична прогресия bₙ² = bₙ₋₁ · bₙ₊₁ (всеки член е средно геометрично на съседите).
📌
Знаменателят q не може да е 0. При q = 1 всички членове са равни. При q > 1 – нарастваща, при 0 < q < 1 – намаляваща.
Пример 1: Дадена е геометрична прогресия 2, 6, 18, 54, ... Намери b₅ и S₄.
1
b₁ = 2, q = 6/2 = 3
2
b₅ = 2 · 3⁵⁻¹ = 2 · 81 = 162
3
S₄ = 2 · (3⁴ − 1) / (3 − 1) = 2 · 80 / 2 = 80
4
Отговор: b₅ = 162, S₄ = 80
Пример 2: Намери знаменателя q, ако b₁ = 4 и b₄ = 32.
1
bₙ = b₁ · qⁿ⁻¹ → b₄ = 4 · q³ = 32
2
q³ = 8 → q = 2
3
Отговор: q = 2
4. Безкрайна геометрична прогресия

При |q| < 1 безкрайната геометрична прогресия има крайна сума. Това е важен матурен факт!

Сума на безкрайна геометрична прогресия (при |q| < 1) S = b₁ / (1 − q)
📌
Условие: |q| < 1 (т.е. −1 < q < 1, q ≠ 0). При |q| ≥ 1 безкрайната сума не съществува (расте до безкрайност).
💡
Приложение: Безкрайни десетични дроби могат да се изразят като дроби. Пример: 0.333... = 3/10 + 3/100 + ... = (3/10)/(1 − 1/10) = 1/3.
Пример: Намери сумата на безкрайната прогресия 12, 4, 4/3, 4/9, ...
1
b₁ = 12, q = 4/12 = 1/3. |q| = 1/3 < 1 ✓
2
S = 12 / (1 − 1/3) = 12 / (2/3) = 12 · 3/2 = 18
3
Отговор: S = 18
5. Типични матурни задачи

На матурата задачите с прогресии обикновено са от следните видове:

🔢
Намери n-ти член: Дадено е a₁ и d (или q). Приложи формулата aₙ = a₁ + (n−1)d или bₙ = b₁ · qⁿ⁻¹.
🔢
Намери сума на n члена: Дадено е a₁, d (или q) и n. Приложи формулата за Sₙ.
🔢
Намери n при дадена сума: Зададена е Sₙ, а₁ и d. Получаваш квадратно уравнение за n, от чиито корени вземаш естественото число.
🔢
Намери разлика или знаменател: Дадени са два члена. Използваш формулата за n-ти член и изразяваш d или q.
Матурна задача: В аритметична прогресия a₁ = 2, d = 3. За кой n е Sₙ = 77?
1
Sₙ = n(2a₁ + (n−1)d)/2 = n(4 + 3(n−1))/2 = n(3n + 1)/2
2
n(3n + 1)/2 = 77 → 3n² + n = 154 → 3n² + n − 154 = 0
3
D = 1 + 4·3·154 = 1 + 1848 = 1849 = 43². n = (−1 + 43)/6 = 7 или n = (−1 − 43)/6 < 0
4
Отговор: n = 7 (Проверка: S₇ = 7·(4 + 18)/2 = 7·11 = 77 ✓)
6. Сравнение: аритметична vs. геометрична прогресия
Характеристика Аритметична прогресия Геометрична прогресия
Правило aₙ₊₁ = aₙ + d bₙ₊₁ = bₙ · q
n-ти член aₙ = a₁ + (n−1)d bₙ = b₁ · qⁿ⁻¹
Сума Sₙ n(a₁ + aₙ)/2 b₁(qⁿ−1)/(q−1)
Среден член aₙ = (aₙ₋₁ + aₙ₊₁)/2 bₙ² = bₙ₋₁ · bₙ₊₁
Безкрайна сума Не съществува S = b₁/(1−q), |q| < 1
7. Най-важното за матурата
📌
aₙ = a₁ + (n−1)d – формулата за n-ти член на АП. Запомни: (n−1), не n!
📌
Sₙ = n(a₁ + aₙ)/2 – сумата на АП е n пъти средното на първия и последния член.
📌
bₙ = b₁ · qⁿ⁻¹ – формулата за n-ти член на ГП. Знаменателят е на степен (n−1)!
📌
S = b₁/(1−q) при |q| < 1 – безкрайна геометрична сума. Ако |q| ≥ 1, сумата не съществува.
📌
Разпознаване: АП – проверявай разлики (a₂−a₁ = a₃−a₂); ГП – проверявай отношения (b₂/b₁ = b₃/b₂).
📌
При намиране на n от Sₙ: получаваш квадратно уравнение. Вземай само положителния целочислен корен!
