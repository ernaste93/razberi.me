Теорема на Питагор
Геометрия · Питагорови тройки · Обратна теорема · ~12 мин четене
Формулата a² + b² = c²

Теорема на Питагор: В правоъгълен триъгълник квадратът на хипотенузата е равен на сумата от квадратите на катетите.

В правоъгълен триъгълник: a² + b² = c² където: a, b — катети (страните, образуващи правия ъгъл) c — хипотенуза (страната срещу правия ъгъл, най-дългата)
📐
Хипотенузата е винаги срещу 90° и е най-дългата страна в правоъгълния триъгълник.
📐
Катетите са двете страни, образуващи правия ъгъл. Те са по-къси от хипотенузата.
Намиране на хипотенуза
Формула: c = √(a² + b²) Пример 1: a = 3, b = 4 c = √(3² + 4²) = √(9 + 16) = √25 = 5 Пример 2: a = 5, b = 12 c = √(5² + 12²) = √(25 + 144) = √169 = 13 Пример 3: a = 6, b = 8 c = √(6² + 8²) = √(36 + 64) = √100 = 10
Намиране на катет
Формула: a = √(c² - b²) Пример 1: c = 13, b = 5 a = √(13² - 5²) = √(169 - 25) = √144 = 12 Пример 2: c = 10, b = 8 a = √(10² - 8²) = √(100 - 64) = √36 = 6 Пример 3: c = 17, b = 15 a = √(17² - 15²) = √(289 - 225) = √64 = 8
Питагорови тройки

Питагорова тройка е наредена тройка от естествени числа (a, b, c), за която a² + b² = c². Тези тройки трябва да се запомнят — много задачи на НВО ги използват.

🔢
(3, 4, 5) — 9 + 16 = 25. Основна питагорова тройка.
🔢
(5, 12, 13) — 25 + 144 = 169. Много честа в задачите.
🔢
(8, 15, 17) — 64 + 225 = 289.
🔢
(6, 8, 10) — умножение на (3,4,5) по 2. 36 + 64 = 100.
🔢
(7, 24, 25) — 49 + 576 = 625.
Обратна теорема на Питагор

Ако в триъгълник с страни a, b, c е изпълнено a² + b² = c², тогава триъгълникът е правоъгълен и правият ъгъл е срещу страната c.

Проверка: Дали триъгълник с страни 9, 40, 41 е правоъгълен? 9² + 40² = 81 + 1600 = 1681 41² = 1681 ✓ ДА, триъгълникът е правоъгълен!
Приложни задачи
🏗️
Стълба на стена: Стълба с дължина 10 м стои с основата 6 м от стената. До каква височина стига? h = √(10² − 6²) = √(100 − 36) = √64 = 8 м.
📦
Диагонал на правоъгълник: Правоъгълник 3×4 → d = √(3² + 4²) = √25 = 5. Диагоналът е хипотенуза на правоъгълния триъгълник.
📍
Разстояние между точки: A(0,0) и B(3,4) → AB = √((3−0)² + (4−0)²) = √(9+16) = √25 = 5.
⬜
Диагонал на квадрат: Квадрат с страна a → d = a√2. При a = 5: d = 5√2 ≈ 7.07.
Най-важното за НВО
📌
a² + b² = c² — формулата на Питагор. Работи само в правоъгълен триъгълник!
📌
Хипотенуза: c = √(a² + b²). Катет: a = √(c² − b²).
📌
Запомни тройките: (3,4,5); (5,12,13); (8,15,17); (6,8,10); (7,24,25).
📌
Обратна теорема: Ако a² + b² = c² → триъгълникът е правоъгълен.
📌
Диагонал на квадрат с страна a: d = a√2.
