Оксиди, киселини, основи и соли — свойства и реакции

Класификация на неорганичните съединения, химични свойства · ~10 мин четене
Оксиди

Оксидите са бинарни съединения на даден елемент с кислород. Делят се на основни оксиди (оксиди на метали с ниска валентност — реагират с киселини и вода), киселинни оксиди (оксиди на неметали и метали с висока валентност — реагират с основи и вода) и амфотерни оксиди (реагират и с двете).

🟦
Основни оксиди: Na₂O, CaO, MgO, FeO, CuO. Реагират с киселини → сол+вода; с вода → основа (ако са разтворими): CaO + H₂O → Ca(OH)₂.
🟥
Киселинни оксиди: SO₂, SO₃, CO₂, N₂O₅, P₂O₅, SiO₂ (с изключения). Реагират с основи → сол+вода; с вода → киселина: SO₃ + H₂O → H₂SO₄.
🟩
Амфотерни оксиди: Al₂O₃, ZnO, Cr₂O₃. Реагират и с киселини, и с основи: Al₂O₃ + 6HCl → 2AlCl₃ + 3H₂O; Al₂O₃ + 2NaOH → 2NaAlO₂ + H₂O.
Киселини

Киселините са вещества, съдържащи водород, заменим с метал, при реакция с метал/основа се получава сол. По Арениус — отделят H⁺ в разтвор. Класификация: безкислородни (HCl, HBr, HI, H₂S, HF, HCN) и кислородни (HNO₃, H₂SO₄, H₃PO₄, H₂CO₃, HClO₄). По сила: силни (HCl, HNO₃, H₂SO₄) и слаби (CH₃COOH, H₂CO₃, H₃PO₄).

Химични свойства на киселините:
Кис. + метал → сол + H₂↑ (само металите пред H в реда на активност)

Кис. + основен оксид → сол + H₂O

Кис. + основа → сол + H₂O (неутрализация)

Кис. + сол → нова кис. + нова сол (ако нова кис. е по-слаба/нелетлива)
Основи (хидроксиди)

Основите съдържат OH⁻ групи. По Арениус — отделят OH⁻ в разтвор. Разтворимите основи се наричат алкали (NaOH, KOH, Ca(OH)₂, Ba(OH)₂). Неразтворими основи: Fe(OH)₃, Cu(OH)₂, Al(OH)₃. Амфотерни: Al(OH)₃, Zn(OH)₂ — реагират и с киселини, и с основи.

Химични свойства на основите:
Осн. + киселинен оксид → сол + H₂O: 2NaOH + CO₂ → Na₂CO₃ + H₂O

Осн. + киселина → сол + H₂O: NaOH + HCl → NaCl + H₂O

Осн. + сол → нова основа + нова сол (ако нова осн. е неразт.)

Неразтв. основи → оксид + H₂O (при нагряване): Cu(OH)₂ → CuO + H₂O
Соли

Солите са йонни съединения от метален катион (или NH₄⁺) и киселинен анион. Образуват се при реакция на киселина с основа (неутрализация) или на метал с киселина. Видове: средни соли (NaCl, CaSO₄, K₂CO₃), кисели соли (NaHCO₃, NaHSO₄ — с H в аниона), основни соли (MgOHCl — с OH в структурата).

🧂
NaCl (натриев хлорид): кухненска сол. Електролит, проводник в разтвор. Получава се: HCl + NaOH → NaCl + H₂O.
🫧
Na₂CO₃ (натриев карбонат): „сода" за пране. Основна сол — хидролизира до основна среда. Na₂CO₃ + H₂O ⇌ NaHCO₃ + NaOH.
🏗️
CaSO₄ · 2H₂O (гипс): при нагряване губи вода и образува хемихидрат (CaSO₄·½H₂O), който при овлажняване отново се втвърдява — използва се в строителството.
🌿
Ca₃(PO₄)₂ (калциев фосфат): основна съставка на костите и зъбите. Слабо разтворим.
Реакции на солите:
Сол + киселина → нова сол + нова кис.: CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

Сол + основа → нова сол + нова осн.: CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄

Сол₁ + Сол₂ → нови соли (при утайка/газ): BaCl₂ + Na₂SO₄ → BaSO₄↓ + 2NaCl

Сол + метал → нова сол + метал: Fe + CuSO₄ → FeSO₄ + Cu

Генетичната връзка между класовете: метал → основен оксид → основа → сол (и обратно). Неметал → киселинен оксид → киселина → сол. Тези трансформации обединяват цялата неорганична химия в логична система.
