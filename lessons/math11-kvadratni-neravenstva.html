Квадратни неравенства и системи
Неравенства · Дискриминанта · Системи · ~10 мин четене
1. Какво е квадратно неравенство?

Квадратното неравенство е неравенство от вида ax² + bx + c > 0, където a ≠ 0. Знакът може да бъде >, <, ≥ или ≤. Числата a, b и c са реални коефициенти, а неизвестното е x.

За разлика от квадратното уравнение, при което търсим конкретни стойности на x, при квадратното неравенство търсим множество от стойности — интервал (или обединение на интервали), за които неравенството е изпълнено.

📌
Основна форма: ax² + bx + c > 0 (или < 0, ≥ 0, ≤ 0), където a ≠ 0.
📌
Решението е множество от реални числа (интервал или обединение на интервали), не единична стойност.
2. Метод на решаване — стъпки

Стандартният метод за решаване на квадратно неравенство следва три ясни стъпки:

1️⃣
Намери корените на квадратния тричлен ax² + bx + c = 0, като използваш формулата: x = (−b ± √D) / 2a, където D = b² − 4ac е дискриминантата.
2️⃣
Начертай схема на знаците — нанеси корените върху числова ос и определи знака на израза ax² + bx + c в трите области.
3️⃣
Запиши решението — избери интервалите, в които знакът съвпада с изисквания от неравенството, и запиши отговора с интервална нотация.
3. Дискриминанта и трите случая

Дискриминантата D = b² − 4ac определя броя на реалните корени и изцяло задава характера на решението.

✅
D > 0 — два различни реални корена x₁ и x₂ (x₁ < x₂). Параболата пресича оста Ox в две точки. Знакът на тричлена се редува: вън от корените има знака на a, между корените — обратния.
〰️
D = 0 — един двоен корен x₀ = −b / 2a. Параболата докосва оста Ox в единствена точка. Тричленът е неотрицателен (ако a > 0) или непозитивен (ако a < 0) навсякъде.
❌
D < 0 — няма реални корени. Параболата не пресича оста Ox. Тричленът запазва знака на a за всички реални x.

Ключовото правило, което трябва да запомниш:

🔑
a > 0: параболата е с клони нагоре — тричленът е отрицателен между корените и положителен извън тях.
🔑
a < 0: параболата е с клони надолу — тричленът е положителен между корените и отрицателен извън тях.
4. Схема на знаците (знакова таблица)

Схемата на знаците (или знаковата таблица) е нагледен инструмент, с който лесно определяме знака на квадратния тричлен в различните интервали. Нека ax² + bx + c има два корена x₁ < x₂ и a > 0:

Интервал: (−∞, x₁) x₁ (x₁, x₂) x₂ (x₂, +∞) Знак на f: + 0 − 0 +

Ако a < 0, знаците са обратни:

Интервал: (−∞, x₁) x₁ (x₁, x₂) x₂ (x₂, +∞) Знак на f: − 0 + 0 −

В точките x₁ и x₂ тричленът е равен на нула. Дали включваме тези точки в решението зависи от знака на неравенството: при строго неравенство (> или <) ги изключваме; при нестрого (≥ или ≤) ги включваме.

5. Примери с подробни решения
Пример 1: x² − 5x + 6 > 0
Стъпка 1 — намираме корените на x² − 5x + 6 = 0 D = (−5)² − 4·1·6 = 25 − 24 = 1 > 0 x₁ = (5 − 1) / 2 = 2 x₂ = (5 + 1) / 2 = 3 Стъпка 2 — схема на знаците (a = 1 > 0) (−∞, 2): + | (2, 3): − | (3, +∞): + Стъпка 3 — търсим където изразът е > 0 Решение: x ∈ (−∞, 2) ∪ (3, +∞)
Пример 2: −x² + 4x − 3 ≥ 0
Стъпка 1 — намираме корените на −x² + 4x − 3 = 0 Умножаваме по −1: x² − 4x + 3 = 0 D = 16 − 12 = 4 > 0 x₁ = (4 − 2) / 2 = 1 x₂ = (4 + 2) / 2 = 3 Стъпка 2 — схема на знаците (a = −1 < 0) (−∞, 1): − | (1, 3): + | (3, +∞): − Стъпка 3 — търсим където изразът е ≥ 0 (включваме краищата) Решение: x ∈ [1, 3]
Пример 3: x² − 4x + 4 < 0 (случай D = 0)
Стъпка 1 — намираме корените D = 16 − 16 = 0 → x₀ = 4 / 2 = 2 (двоен корен) x² − 4x + 4 = (x − 2)² Анализ (x − 2)² ≥ 0 за всяко x, и е равно на 0 само при x = 2. Не съществуват стойности, при които (x − 2)² < 0. Решение Решение: няма решение (празно множество ∅)
Пример 4: 2x² + x + 3 ≤ 0 (случай D < 0)
Стъпка 1 — изчисляваме дискриминантата D = 1² − 4·2·3 = 1 − 24 = −23 < 0 Анализ Тъй като D < 0 и a = 2 > 0, тричленът е строго положителен за всяко x. 2x² + x + 3 > 0 за всяко реално x. Решение Решение: няма решение (празно множество ∅) Забележка: ако неравенството беше ≥ 0, решението щеше да бъде x ∈ (−∞, +∞).
6. Системи от квадратни неравенства

Система от неравенства се решава, като се намерят решенията на всяко отделно неравенство и след това се вземе пресечното множество на получените решения. Казано по-просто: търсим стойностите на x, за които едновременно са изпълнени всички неравенства в системата.

📋
Алгоритъм за система:
1. Реши всяко неравенство поотделно.
2. Нанеси решенията на числова ос.
3. Намери пресечението — това е общото решение.
Пример 5: Система от две неравенства
{ x² − x − 6 < 0 { x² − 4 ≥ 0 Решаване на първото неравенство: x² − x − 6 < 0 D = 1 + 24 = 25 → x₁ = (1−5)/2 = −2, x₂ = (1+5)/2 = 3 a = 1 > 0 → изразът е отрицателен между корените Решение на (1): x ∈ (−2, 3) Решаване на второто неравенство: x² − 4 ≥ 0 x² − 4 = (x−2)(x+2) → корени: x = −2 и x = 2 a = 1 > 0 → изразът е неотрицателен извън корените Решение на (2): x ∈ (−∞, −2] ∪ [2, +∞) Пресичане на двете решения (−2, 3) ∩ ((−∞, −2] ∪ [2, +∞)) = (−2, 3) ∩ [2, +∞) (тъй като −2 не е в отворения интервал) Общо решение: x ∈ [2, 3)

Забележи как нанасянето на двете решения на числова ос прави пресечението веднага видимо. При по-сложни системи тази техника е особено полезна.

7. Интервална нотация — как записваме отговора

Решенията на квадратни неравенства се записват с интервална нотация. Ето основните правила:

( )
Кръгли скоби означават, че крайната точка не е включена — използват се при строго неравенство (> или <) и при ±∞.
[ ]
Квадратни скоби означават, че крайната точка е включена — използват се при нестрого неравенство (≥ или ≤).
∪
Знакът ∪ (обединение) се използва, когато решението се състои от два несвързани интервала, например x ∈ (−∞, −3) ∪ (5, +∞).
8. Обобщение и таблица с правила
📐
Дискриминанта: D = b² − 4ac. Определя броя на реалните корени.
📈
a > 0, D > 0: ax²+bx+c > 0 за x ∈ (−∞, x₁) ∪ (x₂, +∞); ax²+bx+c < 0 за x ∈ (x₁, x₂).
📉
a < 0, D > 0: ax²+bx+c > 0 за x ∈ (x₁, x₂); ax²+bx+c < 0 за x ∈ (−∞, x₁) ∪ (x₂, +∞).
〰️
D = 0: Двоен корен x₀. При a > 0: ax²+bx+c ≥ 0 навсякъде (= 0 само в x₀). При a < 0: ax²+bx+c ≤ 0 навсякъде.
∅
D < 0: Няма реални корени. При a > 0: тричленът е винаги положителен. При a < 0: тричленът е винаги отрицателен.
🔗
Системи: Реши всяко неравенство поотделно, след което вземи пресечното множество (∩) на решенията.
