Аритметична и геометрична прогресия

Редици · Формули · Суми · ~10 мин четене
1. Какво е прогресия?

Прогресията е специален вид числова редица — наредена поредица от числа, при която всеки следващ член се получава по точно определено правило. В математиката изучаваме два основни типа: аритметична прогресия (АП) и геометрична прогресия (ГП). Те се срещат навсякъде — от лихвени сметки и вноски до физика и компютърни алгоритми.

Познаването на тези редици позволява бързо да намерим произволен член или сумата на много членове, без да ги изброяваме един по един.

2. Аритметична прогресия (АП)

Казваме, че редицата a₁, a₂, a₃, … е аритметична прогресия, ако разликата между всеки два последователни члена е постоянна. Тази постоянна се нарича разлика на прогресията и се означава с d.

Определение за АП aₙ₊₁ − aₙ = d = const, за всяко n ≥ 1

Ако d > 0, прогресията е растяща. Ако d < 0, прогресията е намаляваща. Ако d = 0, всички членове са равни — говорим за константна редица.

Примери за АП:

  • 2, 5, 8, 11, 14, … (d = 3, растяща)
  • 10, 7, 4, 1, −2, … (d = −3, намаляваща)
  • 1, 1, 1, 1, … (d = 0, константна)
2а. Общ член на АП

За да намерим n-тия член на АП, не е нужно да изброяваме всички предишни. Достатъчно е да знаем първия член a₁ и разликата d:

Общ член на аритметична прогресия aₙ = a₁ + (n − 1) · d

Тази формула следва директно от определението: всеки нов член добавя d към предишния, следователно n-тият член добавя d точно (n−1) пъти към a₁.

📐
aₙ = a₁ + (n−1)·d — общ член на АП. Знаейки a₁ и d, можем да намерим произволен член мигновено.
🔁
Средна стойност: всеки член на АП е средноаритметичното на двата си съседа: aₙ = (aₙ₋₁ + aₙ₊₁) / 2.
2б. Сума на първите n члена на АП

Намирането на сумата S_n = a₁ + a₂ + … + aₙ е класическа задача. Гаус е открил елегантна формула още като ученик: ако напишем сумата наред и я напишем наред наобратно, всяка двойка членове дава едно и също.

Сума на АП — две еквивалентни форми Sₙ = n · (a₁ + aₙ) / 2   =   n · (2a₁ + (n−1)·d) / 2

Първата форма е удобна, когато знаем първия и последния член. Втората форма е удобна, когато знаем само a₁ и d.

Пример 1: Намерете сумата на първите 20 члена на АП с a₁ = 3 и d = 4.
1
Намираме a₂₀ = a₁ + (20 − 1) · d = 3 + 19 · 4 = 3 + 76 = 79
2
S₂₀ = 20 · (a₁ + a₂₀) / 2 = 20 · (3 + 79) / 2 = 20 · 41 = 820
3
Проверка с втората форма: S₂₀ = 20 · (2·3 + 19·4) / 2 = 20 · (6 + 76) / 2 = 20 · 41 = 820 ✓
Пример 2: 5-тият член на АП е 17, а 12-тият член е 38. Намерете a₁ и d.
1
a₅ = a₁ + 4d = 17 и a₁₂ = a₁ + 11d = 38. Изваждаме: 7d = 21, откъдето d = 3.
2
От a₁ + 4·3 = 17: a₁ = 17 − 12 = 5.
3
Прогресията: 5, 8, 11, 14, 17, 20, 23, 26, 29, 32, 35, 38, … ✓
3. Геометрична прогресия (ГП)

Казваме, че редицата b₁, b₂, b₃, … е геометрична прогресия, ако частното между всеки два последователни члена е постоянно. Тази постоянна се нарича знаменател на прогресията и се означава с q. Изисква се b₁ ≠ 0 и q ≠ 0.

Определение за ГП bₙ₊₁ / bₙ = q = const, за всяко n ≥ 1

Ако |q| > 1, прогресията расте по абсолютна стойност. Ако 0 < |q| < 1, прогресията клони към нула. Ако q < 0, знакът на членовете се редува.

Примери за ГП:

  • 2, 6, 18, 54, … (q = 3, растяща)
  • 64, 16, 4, 1, 1/4, … (q = 1/4, намаляваща)
  • 1, −2, 4, −8, … (q = −2, знакочередуваща)
3а. Общ член на ГП
Общ член на геометрична прогресия bₙ = b₁ · qⁿ⁻¹
📐
bₙ = b₁ · qⁿ⁻¹ — общ член на ГП. Множим b₁ по q точно (n−1) пъти.
🔁
Средна стойност: всеки член на ГП е средногеометрично на двата си съседа: bₙ² = bₙ₋₁ · bₙ₊₁.
3б. Сума на първите n члена на ГП

Формулата за сума на ГП се извежда умно: умножаваме сумата по q и изваждаме от оригиналната. Почти всички членове се съкращават.

Сума на ГП (при q ≠ 1) Sₙ = b₁ · (qⁿ − 1) / (q − 1)

При q = 1 всички членове са равни на b₁, така че S_n = n · b₁.

4. Сума на безкрайна геометрична прогресия

Когато |q| < 1, членовете на ГП клонят към нула и сумата на безкрайно много члена е крайно число. Това е едно от най-изненадващите явления в математиката.

Сума на безкрайна ГП (|q| < 1) S = b₁ / (1 − q)

Тази формула обяснява защо например 0.999... = 1: сумата 9/10 + 9/100 + 9/1000 + … = (9/10) / (1 − 1/10) = (9/10) / (9/10) = 1.

⚠️
Важно: Ако |q| ≥ 1, безкрайната сума не съществува (сумата клони към безкрайност или се люлее).
5. АП срещу ГП — бърза справка
Аритметична прогресия
Следващ член: + d
Общ член: a₁ + (n−1)·d
Сума: n·(a₁ + aₙ)/2
Свойство: aₙ = (aₙ₋₁ + aₙ₊₁)/2
Пример: 2, 5, 8, 11, …
Геометрична прогресия
Следващ член: × q
Общ член: b₁ · qⁿ⁻¹
Сума: b₁·(qⁿ−1)/(q−1)
Свойство: bₙ² = bₙ₋₁·bₙ₊₁
Пример: 3, 6, 12, 24, …
6. Решени примери
Пример 3 — ГП, намиране на членове: В геометрична прогресия b₃ = 12 и b₆ = 96. Намерете b₁ и q.
1
b₃ = b₁ · q² = 12 и b₆ = b₁ · q⁵ = 96.
2
Делим: q³ = 96/12 = 8, откъдето q = 2.
3
От b₁ · 2² = 12: b₁ = 12/4 = 3.
4
ГП: 3, 6, 12, 24, 48, 96, … ✓
Пример 4 — безкрайна ГП: Намерете сумата на безкрайната прогресия 8, 4, 2, 1, 1/2, …
1
b₁ = 8, q = 4/8 = 1/2. Тъй като |q| = 1/2 < 1, безкрайната сума съществува.
2
S = b₁ / (1 − q) = 8 / (1 − 1/2) = 8 / (1/2) = 16.
3
Проверка: 8 + 4 + 2 + 1 + 0.5 + 0.25 + … се приближава все повече до 16. ✓
Пример 5 — смесена задача: Три числа образуват АП. Сумата им е 21, а произведението им е 315. Намерете ги.
1
Нека трите числа на АП са (a−d), a, (a+d). Сумата им е 3a = 21, откъдето a = 7.
2
Произведение: (7−d) · 7 · (7+d) = 315. Опростяваме: 7 · (49 − d²) = 315.
3
49 − d² = 45, откъдето d² = 4, т.е. d = ±2.
4
Трите числа са 5, 7, 9 (или 9, 7, 5). Проверка: 5+7+9 = 21 ✓, 5·7·9 = 315 ✓.
Пример 6 — сума на АП с условие: Намерете сумата на всички естествени числа от 1 до 100, делящи се на 3.
1
Числата са 3, 6, 9, …, 99 — АП с a₁ = 3, d = 3, aₙ = 99.
2
Брой членове: n = (99 − 3)/3 + 1 = 96/3 + 1 = 32 + 1 = 33.
3
S₃₃ = 33 · (3 + 99) / 2 = 33 · 51 = 1683.
7. Важни свойства и бележки
📌
АП — свойство на средата: в АП с нечетен брой членове, средният член е равен на средноаритметичното на всички членове, т.е. на Sₙ/n.
📌
ГП — свойство на симетрия: при ГП произведението на членове, еднакво отдалечени от краищата, е постоянно: b₁ · bₙ = b₂ · bₙ₋₁ = b₃ · bₙ₋₂.
📌
АП и ГП заедно: ако три числа a, b, c са в АП, то 2b = a + c. Ако са в ГП, то b² = a·c. Тези условия се използват в задачи за намиране на числа.
📌
Безкрайна ГП — условие: сумата на безкрайна ГП съществува единствено когато |q| < 1. Формулата S = b₁/(1−q) НЕ важи за |q| ≥ 1.
📌
Практически наблюдения: банкова лихва = ГП (умножение), равни вноски = АП (добавяне). АП расте линейно, ГП расте експоненциално.
8. Обобщение на формулите
🔢
АП — общ член: aₙ = a₁ + (n−1)·d
∑
АП — сума: Sₙ = n·(a₁ + aₙ)/2 = n·(2a₁ + (n−1)·d)/2
🔢
ГП — общ член: bₙ = b₁ · qⁿ⁻¹
∑
ГП — сума (q ≠ 1): Sₙ = b₁·(qⁿ − 1)/(q − 1)
∞
Безкрайна ГП (|q| < 1): S = b₁/(1 − q)
