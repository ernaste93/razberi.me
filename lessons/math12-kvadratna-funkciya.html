Квадратна функция

Функции · Парабола · ~10 мин
Определение

Квадратна функция е всяка функция от вид:

Стандартен вид f(x) = ax² + bx + c

където a, b, c са реални числа и a ≠ 0. Ако a = 0, функцията се превръща в линейна – вече не е квадратна.

Примери: f(x) = x² − 4x + 3, f(x) = −2x² + 1, f(x) = 3x²

Графиката – параболата

Графиката на квадратна функция е парабола. Формата й зависи от знака на водещия коефициент a:

a > 0
∪
Параболата е отворена нагоре. Върхът е минимум.
a < 0
∩
Параболата е отворена надолу. Върхът е максимум.

Колкото по-голямо е |a|, толкова по-„тясна" е параболата. При |a| = 1 е „стандартната" ширина.

Дискриминантата D

Дискриминантата показва колко реални корена (нули) има квадратната функция:

Дискриминанта D = b² − 4ac
✅
D > 0 – два различни реални корена. Параболата пресича x-оста в две точки.
⚡
D = 0 – точно един корен (двоен). Параболата докосва x-оста в един точка – това е върхът.
❌
D < 0 – няма реални корени. Параболата не пресича x-оста (цялата е над или под нея).
Намиране на нулите (корените)

При D ≥ 0 нулите се намират по формулата:

Формула за корени x₁,₂ = (−b ± √D) / 2a
Пример: Намери нулите на f(x) = x² − 5x + 6
1
D = (−5)² − 4·1·6 = 25 − 24 = 1
2
x₁ = (5 + 1) / 2 = 3    x₂ = (5 − 1) / 2 = 2
3
Нулите са x = 2 и x = 3. Параболата пресича x-оста в точки (2, 0) и (3, 0).
Върхът на параболата

Върхът е най-важната точка на параболата – минимумът (при a > 0) или максимумът (при a < 0).

Координати на върха x₀ = −b / 2a     y₀ = f(x₀) = −D / 4a
Пример: Намери върха на f(x) = 2x² − 8x + 3
1
x₀ = −(−8) / (2·2) = 8 / 4 = 2
2
y₀ = 2·4 − 8·2 + 3 = 8 − 16 + 3 = −5
3
Върхът е V(2, −5). Тъй като a = 2 > 0, това е минимум.
Ос на симетрия

Параболата е симетрична спрямо вертикалната права, минаваща през върха:

Ос на симетрия x = −b / 2a = x₀

Това означава, че ако имаш две нули x₁ и x₂, оста на симетрия минава точно по средата между тях: x₀ = (x₁ + x₂) / 2.

Канонична форма

Каноничната (вертексна) форма е особено удобна, защото директно се чете върхът:

Канонична форма f(x) = a(x − x₀)² + y₀

Минимумът/максимумът е y₀ и се постига при x = x₀.

Пример: Запиши f(x) = x² + 4x + 7 в канонична форма
1
x₀ = −4 / (2·1) = −2
2
y₀ = f(−2) = 4 − 8 + 7 = 3
3
f(x) = (x + 2)² + 3   (Върхът е V(−2, 3), минимум)
Монотонност

Монотонността зависи от знака на a и положението спрямо върха:

📈
При a > 0: намалява в (−∞, x₀] и нараства в [x₀, +∞)
📉
При a < 0: нараства в (−∞, x₀] и намалява в [x₀, +∞)
Формули на Виета

Ако x₁ и x₂ са корените на ax² + bx + c = 0, то:

Формули на Виета x₁ + x₂ = −b/a      x₁ · x₂ = c/a

Тези формули са изключително полезни при матурата – позволяват да проверяваш или намираш корени без да решаваш цялото уравнение.

Пример: За x² − 7x + 12 = 0 намери x₁ + x₂ и x₁ · x₂ без да решаваш уравнението.
1
a = 1, b = −7, c = 12
2
x₁ + x₂ = −(−7)/1 = 7
3
x₁ · x₂ = 12/1 = 12
Най-важното
📌
Стандартен вид: f(x) = ax² + bx + c, a ≠ 0
📌
a > 0 → парабола нагоре (∪), минимум; a < 0 → парабола надолу (∩), максимум
📌
Дискриминанта: D = b² − 4ac. D > 0 → два корена; D = 0 → един корен; D < 0 → няма корени
📌
Корени: x₁,₂ = (−b ± √D) / 2a
📌
Връх: x₀ = −b/2a, y₀ = f(x₀)
📌
Канонична форма: f(x) = a(x − x₀)² + y₀
📌
Виета: x₁ + x₂ = −b/a и x₁ · x₂ = c/a
