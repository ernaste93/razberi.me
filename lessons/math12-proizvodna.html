Производна и приложения

Диференциране · Монотонност · Екстремуми · ~15 мин
1. Определение на производната

Производната на функция f(x) в точка x мери скоростта, с която се изменя функцията – с други думи, наклона на допирателната към графиката в тази точка.

Дефиниция f'(x) = lim[h→0] (f(x+h) − f(x)) / h

Геометричен смисъл: стойността f'(a) е наклонът (ъгловият коефициент) на допирателната към графиката на f в точката x = a. Ако f'(a) > 0, допирателната се издига наляво надясно. Ако f'(a) < 0, тя се спуска. Ако f'(a) = 0, допирателната е хоризонтална – типично в екстремум.

2. Правила за диференциране

Не е нужно да смятате всяка производна от дефиниция. Следните правила покриват всички задачи на матурата:

Константа (c)' = 0
Степенна функция (xⁿ)' = n · xⁿ⁻¹
Константен множител (c · f)' = c · f'
Сума / разлика (f + g)' = f' + g'
Произведение (f · g)' = f' · g + f · g'
Частно (f / g)' = (f' · g − f · g') / g²
Тригонометрични функции (sin x)' = cos x   ·   (cos x)' = −sin x
Показателна и логаритмична (eˣ)' = eˣ   ·   (ln x)' = 1/x
3. Примери за намиране на производна
Пример 1: Намери производната на f(x) = 3x³ − 2x + 5
1
Диференцираме член по член, като използваме правилата за сума и константен множител:
f'(x) = (3x³)' − (2x)' + (5)'
2
Прилагаме степенното правило (xⁿ)' = n·xⁿ⁻¹ за всеки член:
(3x³)' = 3·3·x² = 9x²
(2x)' = 2·1·x⁰ = 2
(5)' = 0
3
Резултат: f'(x) = 9x² − 2
Пример 2: Намери производната на f(x) = x² · sin x
1
Разпознаваме произведение на две функции: f = x² и g = sin x. Прилагаме правилото за произведение:
(f · g)' = f' · g + f · g'
2
Намираме производните на всяка функция поотделно:
f' = (x²)' = 2x
g' = (sin x)' = cos x
3
Резултат: f'(x) = 2x · sin x + x² · cos x
4. Монотонност на функция

Производната ни казва дали функцията расте или намалява в даден интервал. Намерете f'(x) и определете знака му:

f'(x) > 0
Функцията е нарастваща в този интервал – графиката се издига отляво надясно.
f'(x) < 0
Функцията е намаляваща в този интервал – графиката се спуска отляво надясно.

Метод: Намерете нулите на f'(x), после проверете знака на f' в характерна точка от всеки интервал между нулите.

5. Екстремуми (минимум и максимум)

Критични точки: точките, в които f'(x) = 0 (или f' не съществува), са кандидати за екстремуми.

Достатъчно условие: ако f' сменя знака си при преминаване през критичната точка x₀:

Локален минимум
f' сменя знака от − към +
(функцията спира да намалява и започва да расте)
Локален максимум
f' сменя знака от + към −
(функцията спира да расте и започва да намалява)
Пример: Намери екстремумите на f(x) = x³ − 3x
1
Намираме производната: f'(x) = 3x² − 3 = 3(x² − 1) = 3(x−1)(x+1)
2
Намираме критичните точки: f'(x) = 0 → x = −1 и x = 1
3
Проверяваме знака на f':
При x < −1: f'(−2) = 3(4−1) = 9 > 0 (нараства)
При −1 < x < 1: f'(0) = −3 < 0 (намалява)
При x > 1: f'(2) = 3(4−1) = 9 > 0 (нараства)
4
Резултат: При x = −1: локален максимум, f(−1) = 2. При x = 1: локален минимум, f(1) = −2.
6. Задачи за оптимизация

Производната е мощен инструмент за намиране на най-добра (максимална или минимална) стойност на величина при зададено условие.

1️⃣
Дефинирай променливите – какво е x, какво търсиш.
2️⃣
Запиши функцията, която ще оптимизираш (площ, обем, цена...).
3️⃣
Използвай ограничението, за да изразиш функцията само с една променлива.
4️⃣
Намери производната и я приравни на нула, за да намериш критичните точки.
5️⃣
Провери дали е max или min чрез знака на f' или стойностите в краищата на дефиниционната област.
Пример: Намери размерите на правоъгълник с периметър 20 и максимална площ.
1
Нека едната страна е x. Тогава другата страна е (10 − x), защото 2x + 2(10−x) = 20.
2
Площта е: S(x) = x · (10 − x) = 10x − x², при 0 < x < 10.
3
Производната: S'(x) = 10 − 2x. Нулираме: 10 − 2x = 0 → x = 5.
4
S''(x) = −2 < 0, следователно x = 5 е максимум. Другата страна: 10 − 5 = 5.
5
Отговор: Правоъгълникът е квадрат 5 × 5, с максимална площ S = 25.
7. Най-важното за матурата
📌
Степенното правило е задължително: (xⁿ)' = n·xⁿ⁻¹. Прилага се за всяка степенна функция – включително и за корени: (√x)' = 1/(2√x).
📌
Производната на константа е 0: (7)' = 0, (−π)' = 0. Много ученици го забравят при по-сложни изрази.
📌
Критична точка ≠ Екстремум: f'(x₀) = 0 е необходимо условие, но не достатъчно. Трябва да проверите смяната на знака на f'.
📌
Монотонност: запомнете – f' > 0 значи нараства, f' < 0 значи намалява. Правете таблица на знаците.
📌
Геометричен смисъл: f'(a) е наклонът на допирателната в точка (a, f(a)). Уравнението й е y − f(a) = f'(a)·(x − a).
📌
При оптимизация: винаги проверявайте физическия смисъл на отговора – дали x попада в допустимия интервал.
