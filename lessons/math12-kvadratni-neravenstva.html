Квадратни неравенства

ax² + bx + c > 0 · Метод на параболата · Знакова схема · ~15 мин
1. Какво е квадратно неравенство?

Квадратното неравенство е неравенство от вида ax² + bx + c > 0 (или с <, ≥, ≤), където a ≠ 0. За разлика от квадратното уравнение, търсим интервал от стойности на x, при които неравенството е изпълнено.

Решаването се основава на графиката на парабола y = ax² + bx + c и нейното положение спрямо оста Ox.

Стандартна форма ax² + bx + c > 0  (или < 0, ≥ 0, ≤ 0)  при a ≠ 0
2. Метод на параболата — алгоритъм

Основният метод за решаване на квадратни неравенства е методът на параболата. Той се прилага в 4 стъпки:

①
Намери дискриминантата: D = b² − 4ac. Тя определя колко реални корена има свързаното уравнение ax² + bx + c = 0.
②
Намери корените x₁ и x₂ (ако D ≥ 0): x₁,₂ = (−b ± √D) / (2a). Приеми x₁ < x₂.
③
Нарисувай параболата мислено (или на лист): ако a > 0, отваря се нагоре; ако a < 0 — надолу.
④
Отчети знака на тричлена по интервали и запиши решението.
3. Знакова схема — ключ към решението

Знаковата схема показва знака на израза ax² + bx + c в различните интервали. Тя зависи от знака на a и от корените x₁, x₂.

a > 0 (парабола нагоре)
− − при x < x₁
+ + при x₁ < x < x₂
+ + при x > x₂
Тричленът е отрицателен между корените и положителен извън тях.
a < 0 (парабола надолу)
+ + при x < x₁
+ + при x₁ < x < x₂
− − при x > x₂
Тричленът е положителен между корените и отрицателен извън тях.
Знакова схема при a > 0 и корени x₁ < x₂
(−∞, x₁)
f(x) < 0
x₁
f = 0
(x₁, x₂)
f(x) > 0
x₂
f = 0
(x₂, +∞)
f(x) < 0

Внимание: Тази схема е за a < 0 (парабола надолу) — тричленът е отрицателен извън корените и положителен между тях. За a > 0 е обратното: отрицателен между корените, положителен извън тях.

4. Решени примери
Пример 1: Реши неравенството x² − 5x + 6 > 0
1
D = 25 − 24 = 1 > 0 → два корена: x₁ = (5 − 1)/2 = 2, x₂ = (5 + 1)/2 = 3
2
a = 1 > 0 → парабола нагоре → тричленът е отрицателен между корените [2, 3] и положителен извън тях.
3
Търсим x² − 5x + 6 > 0 → положителна стойност → извън корените.
4
Решение: x ∈ (−∞, 2) ∪ (3, +∞)
Пример 2: Реши неравенството x² − 5x + 6 ≤ 0
1
Корените са същите: x₁ = 2, x₂ = 3. a = 1 > 0 → парабола нагоре.
2
Тричленът е ≤ 0 (отрицателен или нула) точно между корените и в самите корени.
3
Решение: x ∈ [2, 3] (затворен интервал, включва корените)
Пример 3: Реши неравенството −x² + 4x − 3 > 0
1
a = −1 < 0. D = 16 − 12 = 4. Корени: x₁ = (−4 + 2)/(−2) = 1, x₂ = (−4 − 2)/(−2) = 3.
2
a < 0 → парабола надолу → тричленът е положителен между корените.
3
Решение: x ∈ (1, 3)
5. Специални случаи (D = 0 и D < 0)
📌
D = 0 (един корен x₀): Параболата докосва Ox в един точка. При a > 0: тричленът е ≥ 0 за всички x (нула само при x = x₀). При a < 0: ≤ 0 за всички x.
📌
D < 0 (без реални корени): Параболата не пресича Ox. При a > 0: тричленът е > 0 за всички x (неравенство > 0 има решение ℝ; < 0 — няма решение). При a < 0: обратното.
💡
Трик: Ако забравиш правилото — провери с конкретна стойност! Напр. за x = 0 в тричлена: ако резултатът е положителен, интервалът около 0 е в решението.
Обобщение — при a > 0 D > 0: решение на ax²+bx+c < 0 е (x₁, x₂) ; > 0 е (−∞,x₁)∪(x₂,+∞)
D = 0: ax²+bx+c < 0 няма решение ; > 0 е x ≠ x₀ (т.е. x ∈ ℝ \ {x₀})
D < 0: ax²+bx+c < 0 няма решение ; > 0 е x ∈ ℝ
6. Приложения и задачи от матурата

На матурата квадратните неравенства се срещат в различни контексти:

📐
Параметрични задачи: „За кои стойности на параметъра a уравнението/неравенството има решение?" — прилагаш условие за знака на D.
📐
Задачи с дроби: При неравенство f(x)/g(x) > 0 се прилага знакова схема за числителя и знаменателя поотделно.
📐
Системи от неравенства: Решаваш всяко поотделно и намираш пресечението на решенията.
Задача от матурен тип: За кои x ∈ ℝ е изпълнено x² − 3x − 10 ≤ 0?
1
D = 9 + 40 = 49. Корени: x₁ = (3 − 7)/2 = −2, x₂ = (3 + 7)/2 = 5.
2
a = 1 > 0, парабола нагоре → отрицателна (или нула) между корените.
3
Решение: x ∈ [−2, 5]
7. Най-важното за матурата
📌
a > 0 → парабола нагоре: тричленът е отрицателен между корените x₁ и x₂, положителен извън тях.
📌
a < 0 → парабола надолу: тричленът е положителен между корените, отрицателен извън тях.
📌
При ≥ и ≤: корените се включват в решението (затворени скоби []). При > и <: корените се изключват (отворени скоби ()).
📌
D < 0: ако a > 0, тричленът е винаги положителен (неравенство > 0 е изпълнено за всички x). Ако a < 0 — винаги отрицателен.
💡
Провери с число: след намиране на решението, избери произволно x от решението и провери дали неравенството е изпълнено. Бързо откри грешка!
