Квадратна функция

Парабола и свойства · ~8 мин четене
Квадратна функция

Квадратната функция е f(x) = ax² + bx + c, a ≠ 0. Нейната графика е парабола. Ако a > 0, параболата е с клонове нагоре (минимум); ако a < 0 – с клонове надолу (максимум).

Върхът на параболата е точка V(x₀, y₀), където x₀ = −b/2a и y₀ = f(x₀).

📈
a > 0: Парабола с клонове нагоре – минимум в върха
📉
a < 0: Парабола с клонове надолу – максимум в върха
📍
Връх: x₀ = −b/2a; y₀ = −D/4a, където D = b² − 4ac
🎯
Нули: Корените на ax² + bx + c = 0 са точките, където f(x) = 0
Свойства и четност

Осата на симетрия е вертикалната права x = x₀. Функцията е намаляваща за x < x₀ (при a > 0) и нарастваща за x > x₀.

Нулите на функцията се намират от квадратното уравнение. Знакът на f(x) се определя спрямо нулите и знака на a.

Бърза проверка: Ако D > 0, параболата пресича Ox в 2 точки; D = 0 – в 1; D < 0 – не пресича.
