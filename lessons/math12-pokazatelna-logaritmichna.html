Показателна и логаритмична функция

aˣ · log_a(x) · Свойства · Уравнения · Неравенства · ~20 мин
1. Показателна функция y = aˣ

Показателната функция е от вида y = aˣ, където основата a > 0 и a ≠ 1. Аргументът x е в показателя (степента) на основата.

Примери: y = 2ˣ, y = (1/2)ˣ, y = 10ˣ, y = eˣ.

Показателна функция y = aˣ,   a > 0, a ≠ 1
📐
Дефиниционна област: x ∈ ℝ (всички реални числа) — можем да вземаме aˣ за всяко x.
📐
Множество на стойностите: y ∈ (0, +∞) — показателната функция е винаги положителна, никога не достига 0.
📈
При a > 1 (например 2ˣ): функцията е нарастваща — колкото по-голямо x, толкова по-голямо y.
📉
При 0 < a < 1 (например (1/2)ˣ): функцията е намаляваща — колкото по-голямо x, толкова по-малко y.
💡
Специални стойности: a⁰ = 1 за всяко a, a¹ = a. Графиката минава задължително през точките (0, 1) и (1, a).
a > 1 (нарастваща)
Пример: y = 2ˣ
При x → +∞, y → +∞
При x → −∞, y → 0⁺
Хоризонтална асимптота: y = 0
0 < a < 1 (намаляваща)
Пример: y = (1/2)ˣ
При x → +∞, y → 0⁺
При x → −∞, y → +∞
Хоризонтална асимптота: y = 0
2. Логаритмична функция y = log_a(x)

Логаритмичната функция е обратната на показателната. Ако aʸ = x, то y = log_a(x).

С думи: log_a(x) е степента, на която трябва да вдигнем основата a, за да получим x.

Дефиниция на логаритъм log_a(x) = y  ⟺  aʸ = x
📐
Дефиниционна област: x ∈ (0, +∞) — логаритъм се дефинира само за положителни числа!
📐
Множество на стойностите: y ∈ ℝ — логаритмичната функция приема всякакви реални стойности.
📈
При a > 1: нарастваща функция. log_a(x) расте, когато x расте.
📉
При 0 < a < 1: намаляваща функция. log_a(x) намалява, когато x расте.
💡
Специални стойности: log_a(1) = 0, log_a(a) = 1. Графиката минава през (1, 0) и (a, 1).
3. Свойства на логаритмите

Тези свойства са задължителни за матурата. Те позволяват да опростяваме логаритмични изрази и да решаваме уравнения.

Основни свойства (произведение, частно, степен) log_a(b·c) = log_a(b) + log_a(c)
log_a(b/c) = log_a(b) − log_a(c)
log_a(bⁿ) = n · log_a(b)
Смяна на основата log_a(b) = log_c(b) / log_c(a)

Смяната на основата ни позволява да пресмятаме логаритми с произволна основа чрез логаритми с удобна основа (обичайно lg или ln).

🔢
Натурален логаритъм ln: ln x = log_e(x), където e ≈ 2.718. Използва се в анализа и физиката.
🔢
Десетичен логаритъм lg: lg x = log₁₀(x). Използва се в техниката. lg 10 = 1, lg 100 = 2, lg 1000 = 3.
💡
Полезна формула: log_a(a) = 1 и log_a(1) = 0 — винаги, за всяко допустимо a.
Пример: Изчисли log₂(32).
1
32 = 2⁵
2
log₂(32) = log₂(2⁵) = 5 · log₂(2) = 5 · 1 = 5
Пример: Опрости log₃(27) + log₃(1/3).
1
log₃(27) = log₃(3³) = 3
2
log₃(1/3) = log₃(3⁻¹) = −1
3
Отговор: 3 + (−1) = 2
4. Показателни уравнения

Показателното уравнение съдържа неизвестното x в показателя. Основните методи за решаване:

1️⃣
Привеждане до обща основа: ако aˢ = aᵗ, то s = t. Работи, когато и двете страни могат да се запишат като степени на едно и също число.
2️⃣
Логаритмуване: вземаме логаритъм от двете страни. aˣ = b → x = log_a(b). Работи винаги, когато b > 0.
3️⃣
Заместване: при уравнения от вида a²ˣ − c·aˣ + d = 0 заместваме t = aˣ (t > 0) и получаваме квадратно уравнение.
Пример 1 (обща основа): Реши 2ˣ⁺¹ = 16.
1
16 = 2⁴, значи 2ˣ⁺¹ = 2⁴
2
Основите са равни → показателите са равни: x + 1 = 4
3
Отговор: x = 3
Пример 2 (логаритмуване): Реши 3ˣ = 7.
1
Логаритмуваме с основа 3: log₃(3ˣ) = log₃(7)
2
x = log₃(7) (или чрез смяна на основата: x = lg 7 / lg 3 ≈ 1.77)
Пример 3 (заместване): Реши 4ˣ − 5 · 2ˣ + 4 = 0.
1
Забелязваме 4ˣ = (2²)ˣ = (2ˣ)². Заместваме t = 2ˣ (t > 0)
2
t² − 5t + 4 = 0 → (t − 1)(t − 4) = 0 → t = 1 или t = 4
3
2ˣ = 1 = 2⁰ → x = 0   |   2ˣ = 4 = 2² → x = 2
4
Отговор: x = 0 или x = 2
5. Логаритмични уравнения

При логаритмичните уравнения задължително трябва да намерим ОДЗ (област на допустимите стойности) и да проверим решенията!

ВАЖНО — ОДЗ! В логаритъм log_a(f(x)) е необходимо f(x) > 0. Всяко намерено решение трябва да удовлетворява тези условия. Ако f(x) ≤ 0, решението се отхвърля!
Основен метод log_a(f(x)) = log_a(g(x))  ⟹  f(x) = g(x)  (при ОДЗ)
Пример 1: Реши log₂(x + 3) = 3.
1
ОДЗ: x + 3 > 0 → x > −3
2
Прилагаме дефиницията: x + 3 = 2³ = 8
3
x = 5. Проверка: 5 > −3 ✓
4
Отговор: x = 5
Пример 2: Реши log₃(2x − 1) = log₃(x + 2).
1
ОДЗ: 2x − 1 > 0 и x + 2 > 0 → x > 1/2
2
Логаритмите с еднаква основа → 2x − 1 = x + 2
3
x = 3. Проверка: 3 > 1/2 ✓
4
Отговор: x = 3
6. Показателни и логаритмични неравенства

При неравенствата е критично да внимаваме за знака на монотонност — дали функцията е нарастваща или намаляваща.

КЛЮЧОВО ПРАВИЛО за неравенства:
• При a > 1: aˢ > aᵗ ⟺ s > t (знакът се запазва)
• При 0 < a < 1: aˢ > aᵗ ⟺ s < t (знакът се ОБРЪЩА!)

Същото важи за логаритмите:
• При a > 1: log_a(f) > log_a(g) ⟺ f > g
• При 0 < a < 1: log_a(f) > log_a(g) ⟺ f < g (знакът се ОБРЪЩА!)
Пример 1 (показателно неравенство, a > 1): Реши 2ˣ > 8.
1
8 = 2³, значи 2ˣ > 2³
2
a = 2 > 1 → функцията е нарастваща → знакът се запазва: x > 3
3
Отговор: x ∈ (3, +∞)
Пример 2 (показателно неравенство, 0 < a < 1): Реши (1/2)ˣ > 4.
1
4 = (1/2)⁻², значи (1/2)ˣ > (1/2)⁻²
2
a = 1/2 < 1 → функцията е намаляваща → знакът се ОБРЪЩА: x < −2
3
Отговор: x ∈ (−∞, −2)
Пример 3 (логаритмично неравенство): Реши log₂(x − 1) > 2.
1
ОДЗ: x − 1 > 0 → x > 1
2
log₂(x − 1) > 2 = log₂(4) → при a = 2 > 1 знакът се запазва: x − 1 > 4
3
x > 5. Пресичаме с ОДЗ (x > 1): x > 5
4
Отговор: x ∈ (5, +∞)
7. Най-важното за матурата
📌
log_a(b·c) = log_a(b) + log_a(c) — свойство за произведение. Използва се много при опростяване.
📌
log_a(bⁿ) = n · log_a(b) — степен извадена пред знака. Ключово при решаване на уравнения.
📌
ОДЗ при логаритмични уравнения! Аргументът на логаритъма трябва да е строго положителен. Задължително проверявай!
📌
Знакът се обръща при 0 < a < 1 — при неравенства с намаляваща основа. Много честа грешка на матурата!
📌
Смяна на основата: log_a(b) = log_c(b) / log_c(a). Полезно когато основите са различни.
📌
Показателната функция y = aˣ > 0 винаги — никога не може да е нула или отрицателна. Използва се при ОДЗ на показателни уравнения.
